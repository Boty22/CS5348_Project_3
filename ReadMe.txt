ReadMe.txt

1.  Copy *.java files to a clean directory  (ex.  md Project3)
	ClientWorker.java
	Node.java
	SocketClient.java
	SocketServer.java

2.  Compile all files
	cd Project3
	javac *.java

3.  Run the server

	java SocketServer 2000
 
4.  Run the client

	java SocketClient csgrads1.utdallas.edu 2000
